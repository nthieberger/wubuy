Nunggubuyu Grammar Online

Nunggubuyu is a non-Pama-Nyungan language of Australia, spoken around Numbulwar in Eastern Arnhem Land.

The ISO 693-3 code for the language is nuy; alternate names for the language are Nunggubuju, Wubuy and Yingkwira.

These pages present Jeffrey Heath's description of the language, originally published in three separate volumes, as a web resource.
